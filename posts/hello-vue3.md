# 使用 Vue 3 搭建你的第一个博客页面

通过 Vue 3 和 Vite, 可以非常快速地搭建一个现代化的单页应用作为博客前端界面。

## 功能目标

- 博客文章列表页
- 文章详情页
- 通过 HTTP 请求获取文章数据
- 使用卡片风格展示列表

## 文章列表

本示例中, 文章列表数据来自一个 JSON 接口 `/api/posts.json`, 页面会在挂载时发起一次 HTTP 请求, 然后将结果渲染为卡片列表。

## 文章详情

详情页通过路由参数拿到文章 `id`, 然后:

1. 再次请求列表数据并找到对应文章信息
2. 通过 `GET /posts/{id}.md` 获取 Markdown 内容
3. 使用 `marked` 解析 Markdown
4. 使用 `highlight.js` 渲染代码高亮

你可以继续扩展这个页面, 比如接入真实的后端接口, 或添加分页、搜索等功能。
